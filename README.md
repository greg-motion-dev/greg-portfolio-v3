# Technical Portfolio // Greg 🚀

This is my professional technical portfolio, developed alongside my current client project **Orryana**. The goal of this project is to showcase a modern headless architecture, high-performance web standards, and clean component composition.

## 🛠 Tech Stack

| Tool | Purpose |
| :--- | :--- |
| **React (Vite)** | Frontend library & ultra-fast build tool |
| **Tailwind CSS v4** | Styling (CSS-first engine with zero-runtime variables) |
| **Storyblok** | Headless CMS for dynamic, component-based content |
| **Netlify** | CI/CD, Edge Functions & Hosting |
| **Lucide React** | Optimized iconography |

## 🏗 Key Features & Technical Highlights

* **Headless CMS Integration:** Full integration with the Storyblok Bridge for real-time visual editing and content management.
* **Tailwind v4 Architecture:** Leveraging the new `@theme` configuration and native CSS variables for a future-proof design system.
* **Advanced Component Composition:** Built with the "Thinking in React" mindset—prioritizing reusability, state lifting, and clean prop patterns (inspired by Jonas Schmedtmann’s deep-dive).
* **Project Deep-Dives:** Case studies (including the **Orryana** project) that focus on technical challenges, architectural decisions, and performance metrics.
* **Performance First:** Optimized for 100/100 Lighthouse scores using Vite's bundling efficiency and modern asset loading strategies.

## 🚀 Installation & Local Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Lizzzardking/greg-portfolio-v3.git
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables:**
    Create a `.env` file in the root directory:
    ```env
    VITE_STORYBLOK_TOKEN=your_preview_token_here
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
